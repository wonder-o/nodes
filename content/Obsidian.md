# Obsidian 🔮

Obsidian 是一个强大的知识管理工具，使用 Markdown 格式存储笔记。

## 为什么选择 Obsidian？

- **本地存储** - 数据完全掌握在自己手中
- **双链** - 用 `[[note]]` 连接笔记
- **插件生态** - 丰富的社区插件
- **跨平台** - Windows, Mac, Linux, iOS, Android 全支持

## 推荐插件

- Git - 自动备份到 GitHub
- Templater - 模板功能
- Tag Wrangler - 标签管理
